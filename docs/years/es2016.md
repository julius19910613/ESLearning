# ES2016

ES2016 只有两个核心新增能力，但都很实用，尤其是 `includes()` 已经成为日常代码的高频写法。

## Array.prototype.includes

```js
const roles = ['reader', 'editor', 'admin'];

console.log(roles.includes('editor')); // true
console.log(roles.includes('guest')); // false
console.log([NaN].includes(NaN)); // true
```

- 用于判断数组是否包含某个值。
- 相比 `indexOf(...) > -1`，可读性更直接。
- `includes()` 采用 `SameValueZero` 比较，所以可以正确识别 `NaN`。

## Exponentiation Operator

```js
console.log(2 ** 3); // 8

let value = 3;
value **= 2;
console.log(value); // 9
```

- `**` 等价于 `Math.pow(a, b)`，但语法更短。
- `**=` 是对应的赋值形式。

## 什么时候会用到

- 做成员判断时，优先 `includes()` 而不是 `indexOf()` 魔法数字判断。
- 做幂运算时，优先 `**` 而不是 `Math.pow()`，尤其是表达式嵌套较深时。
