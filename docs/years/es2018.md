# ES2018

ES2018 让异步数据流和对象操作更顺手，同时把 RegExp 的实用性提升了一个台阶。

## Async Iteration

```js
async function* fetchPages() {
  yield ['post-1', 'post-2'];
  yield ['post-3'];
}

async function run() {
  for await (const page of fetchPages()) {
    console.log(page);
  }
}

run();
```

- `async function*` 产生 async iterable。
- `for await...of` 适合消费分页流、事件流、异步队列。

## Object Rest / Spread Properties

```js
const source = { id: 1, title: 'ES2018', draft: false };
const { draft, ...publicPost } = source;
const next = { ...publicPost, year: 2018 };

console.log(publicPost); // { id: 1, title: 'ES2018' }
console.log(next); // { id: 1, title: 'ES2018', year: 2018 }
```

- rest 从对象中“拿走已知字段，收集剩余字段”。
- spread 常用于不可变更新。
- 仍然是浅拷贝，嵌套对象不会自动深复制。

## Promise.finally

```js
Promise.resolve('ok')
  .then(value => {
    console.log(value);
  })
  .finally(() => {
    console.log('cleanup');
  });
```

`finally()` 适合放清理逻辑，例如关闭 loading、释放资源、记录结束状态。

## RegExp 增强

### Named Capture Groups

```js
const re = /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/;
const match = re.exec('2026-06-15');

console.log(match.groups.year); // 2026
```

### Lookbehind

```js
const text = '$199';
console.log(text.match(/(?<=\$)\d+/)?.[0]); // 199
```

### dotAll

```js
console.log(/a.b/.test('a\nb')); // false
console.log(/a.b/s.test('a\nb')); // true
```

### Unicode Property Escapes

```js
console.log(/\p{Script=Han}+/u.test('中文')); // true
```

这些能力让 JavaScript 正则更适合处理结构化文本、多语言文本和复杂提取任务。
